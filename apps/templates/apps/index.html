About K-Nearest Neighbor (KNN)

This classification is using K-nearest neighbor (KNN) algorithm as classification model. KNN is a type of supervised machine learning algorithms. It simply calculates the distance of a new data point to all other training data points. The distance can be of any type e.g Euclidean or Manhattan etc. It then selects the K-nearest data points, where K can be any integer. Finally it assigns the data point to the class to which the majority of the K data points belong.
